HTML

  • HTML, known as Hypertext Markup Language, is one of the three cornerstone languages used in web developement
  • HTML is used to create the structure of a webpage

CSS

  • CCS,known as Cascading Style Sheet, is used to give the content style
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Always remember to link CSS to HTML for styles and changes to take effect

Git

  • Git is a version control system that allows us to manage and keep track of each version of our projects.
  • to access Git, we use command line.
  • Git is different from GitHub because GitHub is used when collaberating with others
  • git status: check what branch we are currently working on
  • git checkout -b branch-name: creates a new branch and switches it

Javascript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer excecutes code in a script
  • Javascript is a text-based programming language used both on the client and server sides that allows us to make the webpage interactive
